# scaleway_iam_saml_certificate (Resource)

Manages SAML certificates for an organization. SAML certificates are used to establish trust between your organization and identity providers. This resource allows you to manage SAML certificates in your Scaleway organization.



## Example Usage

```terraform
### Enable IAM SAML for the default organization and add a signing certificate

resource "scaleway_iam_saml" "test" {
}

resource "scaleway_iam_saml_certificate" "main" {
  saml_id = scaleway_iam_saml.test.id
  type    = "signing"
  content = "<signing certificate generated by your Identity Provider>"
}
```
