# scaleway_vpc_public_gateway_refresh_ssh_keys_action (Action)

The [`scaleway_vpc_public_gateway_refresh_ssh_keys_action`](https://registry.terraform.io/providers/scaleway/scaleway/latest/docs/actions/vpc_public_gateway_refresh_ssh_keys_action) action is helpful to trigger a refresh of the SSH keys for a given Public Gateway .

Refer to the VPC Public Gateways [documentation](https://www.scaleway.com/en/docs/public-gateways/) and [API documentation](https://www.scaleway.com/en/developers/api/public-gateways/) for more information.
